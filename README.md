# API de Gestion de Sondages et Analyse par Machine Learning

### Description du projet
Ce projet consiste en une API REST développée avec Flask, conçue pour automatiser la gestion des données de sondages. Elle intègre des modèles d'apprentissage automatique pour fournir des analyses avancées, notamment la segmentation (clustering) et des recommandations personnalisées basées sur les réponses collectées.

### Fonctionnalités techniques
- **Collecte de données** : Endpoints dédiés à l'importation et à la structuration des réponses de sondages.
- **Clustering** : Utilisation d'algorithmes de Machine Learning pour regrouper les profils de répondants.
- **Moteur de recommandation** : Système basé sur les modèles prédictifs pour orienter la prise de décision.
- **Analyse statistique** : Prétraitement et exploitation des variables quantitatives et qualitatives.

### Stack Technique
- **Langage** : Python 3.x
- **Framework API** : Flask
- **Machine Learning** : Scikit-learn, Pandas, NumPy
- **Analyse de données** : Matplotlib / Seaborn (pour les visuels de recherche)
